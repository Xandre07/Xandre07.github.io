Frente-a-Frente

Os dois processadores foram colocados para comparação no site cpu-world.com.

Os gráficos abaixo demonstram a diferença entre as características mais importantes do AMD Ryzen 5 3400G e do Ryzen 3 3200G.
Esses recursos, junto com um número IPC (instruções por ciclo), determinam o desempenho de uma CPU. O gráfico "Número de núcleos / threads" mostra o número de núcleos (área mais escura). A área mais clara no gráfico corresponde ao número de threads extras, fornecidos pela tecnologia Hyper-Threading.
O gráfico de "Frequência operacional" usa a cor escura para a frequência base, enquanto a área mais clara é para a frequência adicional, fornecida pelo recurso Turbo.
A área mais escura no gráfico "Cache On-chip" é para o cache L2 On-chip. A área mais clara é para o cache L3. Embora o cache L3 seja geralmente maior do que o cache L2 On-chip, ele é sempre mais lento.

Vantagens do AMD Ryzen 3 3200G

  • Nenhuma

Vantagens do AMD Ryzen 5 3400G

  1. Uma das características distintivas do processador é a tecnologia Hyper-Threading.
    O Hyper-Threading permite que cada núcleo da CPU execute duas threads simultaneamente.

  2. A frequência de operação deste processador é maior do que a frequência do processador AMD 3200G, como resultado, o AMD 3400G pode executar a maioria dos aplicações com mais rapidez.
